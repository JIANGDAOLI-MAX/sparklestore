# DESIGN.md

## 品牌与视觉方向
- 小红书风格假睫毛种草Live图生成器
- 主色调：红粉渐变（from-red-500 to-pink-500）
- 背景色：随机莫兰迪色系（低饱和度高级感）
- 风格关键词：甜酷、Y2K、高级感、种草感

## Design Tokens
### 色彩
- 主色：红粉渐变 (#ff4757 → #ff6b81)
- 背景色：莫兰迪色系随机选取（如 #B8A9C9, #A3B5A6, #C4A882, #9BB5CE 等16种）
- 文字：白色为主（在莫兰迪背景上），带文字阴影增强可读性
- 装饰：白色半透明蕾丝边角装饰

### 组件观感
- Live图卡片：莫兰迪底色 + 蕾丝装饰 + 对角线布局（产品图左上，眼部图右下）
- 按钮：圆角胶囊型，渐变填充
- 标签：圆角胶囊型，半透明白色毛玻璃背景
- 整体圆角偏大（rounded-2xl/3xl），视觉柔和

### Live图布局
- 左上区域：产品图 + 右侧产品信息（名称、效果描述、亮点标签）
- 右下区域：上眼效果图（带"上眼效果"标签）+ 左侧文案（标题、副标题、正文摘要）
- 底部：标签栏 + 水印
- 四角：白色蕾丝花纹装饰
